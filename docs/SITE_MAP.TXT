

SECTIONS: SECTIONS WILL BE THE FIRST LAYER OF THE LEFT SIDE NAVIGATION BAR. CLICKING ON A SECTION WILL DIRECT USERS TO THE DASHBOARD OF THAT SECTION. EACH SECTION WILL THEN HAVE A LAYER OF PAGES THAT ARE GROUPED INTO THAT SECTION. 
    SECTION LIST: 
        USER: THE USER SECTION WILL CONTAIN ALL SELF VIEW AND SELF SERVERICE PAGES.
        STAFF: THE STAFF SECTION WILL CONTAIN ALL OF THE J-1 / PERSONNEL MANAGEMENT PAGES.
        RECRUITING: THE RECRUITING SECTION WILL CONTAIN ALL OF THE RECRUITING PAGES.
        TRAINING: THE TRAINING SECTION WILL CONTAIN ALL OF THE J-7 / TRAINING PAGES.
        ADMIN: THE ADMIN SECTION WILL CONTAIN ALL OF THE SYSTEM ADMINISTRATION PAGES.
    SECTION KEYS:
        user_
        staff_
        recruiting_
        training_
        admin_
